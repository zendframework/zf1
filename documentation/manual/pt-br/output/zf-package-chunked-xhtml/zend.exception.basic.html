Uso básico

Zend_Exception é a classe base para todas as exceções lançadas pelo Zend Framework. Esta classe estende a parte essencial da classe Exception do PHP.

Example #1 Tratando todas as exceções do Zend Framework

  1. try {
  2.     // seu código
  3. } catch (Zend_Exception $e) {
  4.     // faça alguma coisa
  5. }

Example #2 Tratando as excepções lançadas por um componente específico do Zend Framework

  1. try {
  2.     // seu código
  3. } catch (Zend_Db_Exception $e) {
  4.     // faça alguma coisa
  5. }
